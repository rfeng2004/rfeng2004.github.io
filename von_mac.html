VON Installation Tutorial (Mac)

Note: All credits go to Evan Chen for creating VON, I simply made minor edits so that it would work on Mac.

Prerequisites

  • TeX Live: This tutorial is not guaranteed to work with other TeX distributions.
  • vim: I believe vim may be automatically installed on Macs. To check, open a terminal window and see if the command vim --version gives errors or prints a helpful message.
  • evince: This prerequisite is not strictly required but I highly recommend installing evince as it is a great pdf viewer. You can install evince on Macs through Homebrew. After installing Homebrew, run brew install evince to install evince (this may take a while so be patient). If you decide not to use evince, there is an extra step needed when the .latexmkrc file is downloaded.

Downloads

If you already have some of these files, it's not necessary to download them again.
  • VON source files for Mac
  • Style files by Evan (right click and press "Save As..." and delete the ".txt" extension before saving):
    • evan.sty
    • von.sty
  • Some .asy packages that aren't installed automatically with TeX Live (thanks Evan for hosting them :P):
    • cse5.asy
    • olympiad.asy
  • Open a new terminal window and run the following command to download a copy of .latexmkrc: curl -o .latexmkrc https://rfeng2004.github.io/files/latexmkrc.
    • If you opted not to use evince (or want to use a pdf viewer different from evince), use the command vim .latexmkrc and press "i", then replace "evince" with "open" on the third line for Preview the name of your desired pdf viewer (for example zathura). Hit escape and type ":x" then enter to save your changes.

Installation Instructions

  1. Find out what your home directory is. For me it's "/Users/raymondfeng". To find out your own, open a new terminal window and use the command pwd.
  2. Move "evan.sty" and "von.sty" into the directory "/Users/[complete your home directory]/Library/texmf/tex/latex", creating these directories if they did not already exist.
  3. Move "cse5.asy" and "olympiad.sty" into the directory "/usr/local/texlive/[your year of TeX Live]/texmf-dist/asymptote". You should not need to make new directories along the way this time; however, to move the files into the directory you may need superuser permission (i.e. you will need to enter your password).
  4. Unzip the zip file containing the VON source code.
  5. Open the "von" folder which came out of the zip file. Navigate to "von/controller/po.py", open up the file using IDLE or TextEdit, and change "Raymond Feng" to your name :).
  6. Create a folder called ".von" in your home directory. The easiest way to make this folder is to open a new terminal window and enter the command mkdir .von.
  7. Move the "von" folder that came out of the zip file into the ".von" folder you just created.
  8. In a new terminal window enter the command vim .bash_profile. Hit "i" and you should be able to edit the file. Paste

    into the file (if this file existed before, don't change any existing lines, just add the above code into the file). Hit the escape key and then type ":x" and hit enter to save your changes.
  9. In a new terminal window, enter the command cd .von/von and then run the commmand sudo pip3 install -r requirements.txt. Then, run sudo pip3 install pyyaml. For the last 2 commands, you will have to input your password.
  10. In a new terminal window, enter the command mkdir Documents/von (this is where all TeX files created by VON will live, you can create subfolders to organize entries by contest/year, etc.).
  11. All set! Open a new terminal window and run the command von. To get started with using VON, read Evan's tutorial here. To quit VON, press control+D (not command+D).
